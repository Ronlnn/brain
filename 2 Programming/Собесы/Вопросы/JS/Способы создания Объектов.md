1) Литеральная нотация
```js
let person = {
	name: "Arthur",
	age: 23
};
```
2) Функция-конструктор
```js
function Person(name,age){
	this.name = name;
	this.age - age;
};
let arthur = new Person("Arthur", 23);
```
3) Object.create();
```js
let personProto = {
	sayHello: function(){
	console.log("Hello!")}
};
let arthur = Object.create(personProto);
arthur.name = "Arthur";
arthur.age = 23;
```
4) Class
```js
class Person {
	constructor(name,age){
		this.name = name;
		this.age = age;
	}
}
```
